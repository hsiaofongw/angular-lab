# AngularLab

Angular 组件学习项目．

## 目录结构

`src` 目录下的目录结构：

```
src
├── app
│   ├── demos
│   │   ├── logic-operator-demo
│   │   └── rating-bar-demo
│   └── public-components
│       ├── logic-input
│       └── rating-bar
├── assets
└── environments
```

### `src/app/demos` 目录

存放 Demo 的目录，每一个 Demo 着重于演示不多的几个组件，每个 Demo 以一个 NgModule 的形式生成．

### `src/app/public-components` 目录

存放公共组件的目录，每一个公共组件以一个 NgModule 的形式生成．
